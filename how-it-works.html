⚙️ How It Works

1

User Sends Message

Customer types a message in the chat interface. The message is sent to the Flask backend via API.

2

ML Analysis

The message goes through:

  • 🔍 Fraud Detection - Checks if order is suspicious
  • 💡 Recommendations - Suggests related products
  • 😊 Sentiment Analysis - Detects customer emotion
3

AI Processing

Groq LLaMA 3.3 model generates a contextual response based on sentiment and analysis.

4

Response Delivery

The response is returned to the UI with appropriate emoji and sentiment-aware messaging.
